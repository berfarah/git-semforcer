# GitSemforcer

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'git-semforcer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install git-semforcer

## Usage

```
git semforcer init
# makes a pre-commit hook that enforces semantic versioning

git semforcer destroy
# removes said hook
```
